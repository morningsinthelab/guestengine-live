## COPY SYSTEM NOTES

### Syncopate Headline Character Guide
Syncopate at bold display size is approximately 1.2x wider than standard serif fonts. At 1400px viewport, keep each line under 22-24 characters for safety. All headlines above have been written and verified within this constraint.

### The Gold Accent Rule
The `<span class="gold">` line should always carry the **resolution, contrast, or twist** — never the setup. The gold is the payoff.

### AI Search Engine Mentions
Per brief, all verticals explicitly name: **Google, Perplexity, ChatGPT, Gemini**. Authors page additionally names **Amazon**. Consultants and coaches name **LinkedIn** explicitly in the body copy.

### The $497 Comparison Framework
Each vertical's pricing line is calibrated to their specific spend context:
- **Authors** → PR firm hourly rate
- **Coaches** → Social media manager monthly cost
- **Founders** → PR agency hourly rate
- **Real Estate** → One day of Google Ads
- **Podcasters** → One Facebook Ads campaign
- **Consultants** → Cold outreach sequence cost

### Tone Calibration
All copy is written at a bar-conversation register — direct, specific, no corporate padding. Problems are named before they're articulated. The reader should feel understood before they're sold.

### The Core Reframe (Use Internally)
- **Old frame:** Get booked → Get content → Get visible
- **New frame:** Build your professional identity → The appearance is the activation event → The profile compounds forever
